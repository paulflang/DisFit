`calc` documentation
=======================


`calc` emcompasses two tools:

* Command line interface: See :numref:`cli`.
* Python API: See :numref:`python_api`.

`calc` can tell you if a number is prime.

To determine whether a number is prime from the command line tyme.::
  
    calc isprime <number>

To determine whether is number is prime in python type.::
  
    calc.isPrime(n)

where n is a number
